GradleEnterpriseApi

interface GradleEnterpriseApi(source)

Gradle Enterprise API client. API endpoints are grouped exactly as in the Gradle Enterprise API Manual:

  • buildsApi

  • buildCacheApi

  • metaApi

  • testDistributionApi

Create an instance with newInstance:

val api = GradleEnterpriseApi.newInstance()
api.buildsApi.getBuilds(...)
