Information about the local build cache used in the build, if it was configured in the build. May be null if the local build cache is not configured.

Parameters

isEnabled

Whether the local build cache was enabled.

isDisabledDueToError

Whether the local cache was disabled due to an error occurring in loading or storing local build cache entries.

isPushEnabled

Whether pushing to the local build cache was enabled. May be null if the local build cache is disabled.

directory

Location of the local build cache. Can be relative or absolute depending on its value. May be null if the local build cache is disabled.
