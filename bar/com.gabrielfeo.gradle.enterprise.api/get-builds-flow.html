Gets builds on demand from the API, in as many requests as necessary. It allows for queries of any size, as opposed to GradleEnterpriseApi.getBuilds which is limited by the API itself to 1000.

  • Will request from the API until results end or an error occurs.

  • Use Sequence.take and similar functions to stop collecting early.

  • A subset of getBuilds params are supported
